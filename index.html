Project 1

Predicting Fraud Detection in Online Transactions using Data Mining techniques

  • Statistically analyze and visualize possible factors correlated to fraud commitment

  • Used Logistic Regression, Random Forest, SVM, Neural Network to build fraud detection models

  • Used K-fold Cross Validation to evaluate each model

  • Measures of accuracy include Average Accuracy, Average Precision, Macro F-1 score
